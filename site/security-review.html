Security review notes

Built to be easy to review before work use.

DictaHue is designed as a local dictation app: voice capture, transcription, logs, and light structure stay on the Mac unless the user explicitly exports diagnostics.

Permissions

Microphone for voice. Accessibility/Input Monitoring for hotkey and paste. No Screen Recording in V0.

Network model

No cloud transcription in the shipped product path. Local servers bind to loopback only.

Diagnostics

Local by default. Raw audio and transcripts are excluded unless the user chooses to include them.

Release integrity

Public builds must be Developer ID signed, notarized, stapled, and published with SHA-256 checksums.

Before public release, DictaHue also needs the signed DMG, notary profile, embedded runtime audit, dependency/SBOM review, third-party model notices, and a clean-machine install test.
