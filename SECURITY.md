# Security Policy

## Supported Version

RambleFix is pre-1.0. Security fixes target the latest public release.

## Reporting

Please open a private security advisory on GitHub for vulnerabilities.

Do not include private audio, transcripts, API keys, or company data in public issues. Use the app's diagnostics export and review the bundle before sharing it.

## Product Security Promise

- Dictation runs locally by default.
- No signup is required.
- Diagnostics are user-controlled.
- Raw audio and raw transcripts are excluded from diagnostics unless the user explicitly includes them.
- V0 does not request Screen Recording permission.
